## 🛠️ INSTALACIÓN

### 🐧 Linux / Mac
```bash
git clone [https://github.com/tu-usuario/turismo-quillabamba.git](https://github.com/tu-usuario/turismo-quillabamba.git)
cd turismo-quillabamba
python3 -m venv entorno
source entorno/bin/activate
pip install -r requirements.txt
cd config
python3 manage.py migrate
python3 manage.py runserver
```






### 🪟 Windows (CMD)
```cmd
